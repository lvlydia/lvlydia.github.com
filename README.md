# 重构Vue.js中文社区

## 前言
基于 vue2.0 重构的 Vue.js 中文社区，感谢 Vue.js 中文社区提供的API  
线上[预览地址](https://xypecho.github.io/vueCommunity)  
二维码扫描预览地址

## 技术栈
vue2.0 + vue-cli + vue-resource + vue-router + webpack + ES6 +stylus  
## 项目运行
```
git clone https://github.com/xypecho/vueCommunity.git
cd vueCommunity
npm install
npm run dev
```
